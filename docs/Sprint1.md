# User Stories 
user story1:**create item** As a user i want to add food items with names and quantities so I can track my stock.
user story2:**Delete Item:** As a user,I want to remove items I no longer have so my list stays accurate.
user story3:**Edit Item:** As a user,I want to update the quantity of an item so I don't have to delete and re-add it.
user story4:**Expiration Dates:** As a user,I want to add an expiration date to items to prevent food waste.
user story5:**Categorization:** As a user,I want to tag items (e.g., "Dairy", "Grains") to keep my pantry organized.
user story6:**Low Stock Alerts:** As a user,I want to see a warning when an item's quantity is low.
user story7:**Basic Shopping List:** As a user,I want to move "Zero" quantity items to a shopping list automatically. 
user story8:**Manual Shopping Add:** As a user,I want to manually add "to-buy" items to a list. 
user story9:**Create Household:** As a user,I want to create a "Household" group to share my pantry with others. 
user story10:**Join Household:** As a roommate,I want to join an existing household using a code.
